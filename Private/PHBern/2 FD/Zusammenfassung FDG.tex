\section{Planung von kompetenzorientiertem Geografieunterricht}
\subsection{Modell der didaktischen Rekonstruktion (MDR)}


\textbf{Ziel:} Wissenschaftliche Inhalte lern- und lehrbar machen, indem Vorwissen der Lernenden modifiziert und angereichert wird. Schülervorstellungen sind Schlüssel für erfolgreichen Lernprozess.

\begin{itemize}
  \item \textbf{Fachliche Klärung:}
        \begin{itemize}
          \item Analyse des fachlichen Inhalts auf elementare Konzepte, Begriffe, Prozesse, Theorien, Methoden
          \item Entwicklung einer didaktisch sinnvollen Sachstruktur
        \end{itemize}
  \item \textbf{Erfassung der Lernendenperspektiven:}
        \begin{itemize}
          \item Analyse der Voraussetzungen der Lernenden (Vorstellungen, Interessen, Einstellungen, Lernvoraussetzungen)
          \item Berücksichtigung kognitiver und affektiver Aspekte
          \item Schülervorstellungen als Ausgangs- und Bezugspunkt
        \end{itemize}
  \item \textbf{Didaktische Strukturierung:}
        \begin{itemize}
          \item Verbindung von fachlicher Klärung und Lernendenperspektive
          \item Einbezug fachlicher, überfachlicher, lebensweltlicher, gesellschaftlicher, wissenschaftlicher und ethischer Aspekte
          \item Umgestaltung des Lernstoffs im Hinblick auf Verstehensmöglichkeiten und -schwierigkeiten der Lernenden
        \end{itemize}
\end{itemize}

\subsection{Planungsprozess}

Die Unterrichtsplanung im Geografieunterricht folgt keiner festen Reihenfolge, sondern ist ein iterativer Prozess. Sie orientiert sich an sechs zentralen Dimensionen, die miteinander vernetzt sind:

\begin{itemize}
  \item \textbf{Voraussetzungen:} äussere Bedingungen, Sozialverhalten, Vorkenntnisse.
  \item \textbf{Sachanalyse:} Recherche, Themenauswahl nach MDR, Strukturierung der Inhalte.
  \item \textbf{Didaktische Analyse:} gesellschaftliche, fachliche, schüler- und raumbezogene Relevanz.
  \item \textbf{Lernziele:} aus dem Lehrplan, für den eigenen Unterricht, inkl. Kompetenzen und Transfer.
  \item \textbf{Methodische Analyse:} Methodenwahl, Medieneinsatz, Ergebnissicherung.
  \item \textbf{Verlaufsplanung:} Unterrichtsphasen, Lernaufgaben, Methoden, Medien.
\end{itemize}

Diese Aspekte unterstützen eine kohärente Vorbereitung, Durchführung und Auswertung des Unterrichts.

\section*{Herausfordernde Lernaufgaben}

\textbf{Herausfordernde Lernaufgaben} $\rightarrow$ zentral für \textbf{kompetenzorientierten Geografieunterricht}, da sie aktive, nachhaltige Lernprozesse anregen.

Zentrale Merkmale
\begin{itemize}
  \item \textbf{Situationsgebundenheit:} Die Aufgaben sind in eine \textbf{relevante Lebenswelt- oder Problemsituation} eingebettet.
  \item \textbf{Offenheit:} Es gibt \textbf{mehrere Lösungswege oder Perspektiven}.
  \item \textbf{Problemorientierung:} Es steht eine \textbf{relevante, komplexe Fragestellung} im Zentrum.
  \item \textbf{Fachliche Tiefe:} Die Aufgaben beziehen sich auf \textbf{zentrale geografische Konzepte, Modelle oder Methoden}.
  \item \textbf{Reflexionsförderung:} Lernende sollen \textbf{ihre Ergebnisse, Denkwege und Annahmen reflektieren}.
  \item \textbf{Anforderungsvielfalt:} Es werden \textbf{mehrere Kompetenzbereiche} (z.\,B. Fachwissen, Kommunikation, Urteilen) angesprochen.
\end{itemize}

Didaktische Zielsetzung
\begin{itemize}
  \item \textbf{Kognitive Aktivierung} und \textbf{vernetztes Denken} fördern.
  \item \textbf{Transferleistungen ermöglichen}: Gelerntes auf neue Kontexte anwenden.
  \item \textbf{Urteilskompetenz} fördern, z.\,B. durch raumbezogene Dilemmata.
  \item \textbf{Handlungskompetenz} entwickeln, etwa durch Planspiele oder Zukunftswerkstätten.
  \item \textbf{Werteorientierung} einbauen, z.\,B. bei ökologischen oder ethischen Fragen.
\end{itemize}

Didaktische Gestaltung
\begin{itemize}
  \item \textbf{Einbettung in eine Lernsituation}: Aufgaben sind Teil eines \textbf{vollständigen Lernprozesses} (Einstieg, Material, Auswertung, Transfer).
  \item \textbf{Klare Arbeitsaufträge}: \textbf{Ziel, Vorgehen und Bewertungskriterien} sind transparent.
  \item \textbf{Differenzierung}: Aufgaben bieten \textbf{Zugänge für verschiedene Leistungsniveaus} und Lernwege.
\end{itemize}

Beispielhafte Aufgabenformate
\begin{itemize}
  \item \textbf{Mystery-Aufgaben}
  \item \textbf{Expertenpuzzle mit Perspektivenwechsel}
  \item \textbf{Fallanalysen mit Beurteilungsaufträgen}
  \item \textbf{Planspiele zu Raumkonflikten}
  \item \textbf{Projektorientierte Lernaufgaben}
\end{itemize}

\subsection{Summative Beurteilung}
\begin{itemize}
  \item Ziel: Abschliessende Ermittlung des Lernstands – inwieweit wurden Basiswissen und Kompetenzen erworben.
  \item Leistungsfeststellung soll objektiv, nachprüfbar, valide, reliabel und ökonomisch sein:
        \begin{itemize}
          \item \textbf{Objektivität:}
                \begin{itemize}
                  \item Gleiche Bedingungen für alle (z.B. frühzeitige Bekanntgabe von Termin/Inhalten, Sitzordnung, keine individuellen Hilfen)
                  \item Transparente Punkteverteilung und festgelegtes Auswertungsschema
                  \item Teilaufgaben klassenübergreifend korrigieren
                  \item Aufgaben und Punkteverteilung auf die drei Anforderungsbereiche (Wissen, Anwendung, Weiterentwicklung) abstimmen
                  \item Einfache Einstiegsfragen
                \end{itemize}
          \item \textbf{Nachprüfbarkeit:}
                \begin{itemize}
                  \item Transparente Korrektur (Korrekturzeichen, Punktevergabe, Besprechung)
                  \item Bewertungskriterien stimmen mit Aufgabenstellung überein
                \end{itemize}
          \item \textbf{Validität:}
                \begin{itemize}
                  \item Prüfung misst, was sie messen soll (Übereinstimmung mit Lernzielen)
                  \item Eindeutige, verständliche Aufgabenstellung (bekannte Operatoren)
                  \item Verschiedene Aufgabentypen und Inhalte prüfen
                  \item Angemessener Zeitumfang
                \end{itemize}
          \item \textbf{Reliabilität:}
                \begin{itemize}
                  \item Aufgabenstellung eindeutig und verständlich
                  \item Unabhängige Einzelaufgaben
                  \item Aufgaben und Punkteverteilung auf die drei Anforderungsbereiche
                  \item Verschiedene Inhalte prüfen
                \end{itemize}
          \item \textbf{Ökonomie:}
                \begin{itemize}
                  \item Vertretbarer Arbeits- und Zeitaufwand
                  \item Inhaltlich eingegrenzte Fragestellung
                  \item Bekannte Punktevergabe
                \end{itemize}
        \end{itemize}
\end{itemize}
