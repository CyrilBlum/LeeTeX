\section{Auftrag}
Lesen Sie im Buch ``Hello World'' von Hannah Fry die folgenden Kapitel: 

\begin{enumerate}
	\item ``Einleitung''
	\item ``Macht''
	\item ``Daten''
	\item Ein Kapitel zu einem Thema  Ihrer Wahl (s. Abschnitt \ref{sec:themen}).
\end{enumerate}
Selbstverständlich dürfen Sie auch weitere Kapitel aus dem Buch lesen. 

Machen Sie sich Notizen zu jedem Kapitel (worum geht es? Was ist Ihnen geblieben? Was verstehen Sie / nicht? etc.). 

Im Anschluss an die Lektüre werden Sie folgende Abgaben erstellen:

\begin{enumerate}
	\item \textbf{Gruppenarbeit}: Ein \textbf{Fact Sheet} zum Kapitel Ihrer Wahl
	\item \textbf{Einzelarbeit}: \begin{enumerate*}
		\item Eine \textbf{Präsentation} Ihres Unter-Themas, \item eine \textbf{Zusammenfassung} zum Unterthema Ihrer Wahl, sowie \item ein \textbf{Peer Review} für eine Zusammenfassung einer anderen Person.
	\end{enumerate*} 
\end{enumerate}

Die Gruppenarbeit (Punkt 1), die Zusammenfassung (Punkt 2b) sowie das Peer Review (Punkt 2c) zählen gemeinsam als \textit{eine schriftliche Note}. Dabei tragen Punkte 1 und 2b jeweils  die Hälfte zur Note bei, Punkt 2c gibt bei Nicht- oder nachlässiger Erledigung einen Abzug auf die Gesamtnote. Die Präsentation macht die mündliche Note des Semesters aus und zählt wie eine \textit{halbe schriftliche Prüfung}.

Die Anforderungen für jede dieser Abgaben sind in den folgenden Abschnitten kurz erklärt.

\subsection{Gruppen-Auftrag}
Bilden Sie Dreier- oder Vierergruppen nach Kapitel, das Sie präsentieren. Pro Gruppe bearbeiten Sie also genau eines der sieben Kapitel. Jedes Kapitel wird somit auch nur von genau einer Gruppe bearbeitet. 

\subsubsection*{Austausch in der Gruppe}
Sie tauschen sich in der Gruppe zu folgenden Themen aus:

\begin{itemize}
	\item \textbf{Gedankenaustausch zum Gelesenen}: Worum ging es? Was ist Ihnen \textit{persönlich} geblieben?
	\item \textbf{Organisation des Auftrags}: evtl. Fragen zum Vortrag, Ideen-Sammlung, Vorgehen, Aufgabenverteilung 
\end{itemize}
Notieren Sie sich, was Sie diskutiert haben.
\subsubsection*{Fact Sheet}
Sie erstellen in der Gruppe zu Ihrem Thema ein Fact Sheet. Es besteht aus 1-2 A4-Seiten und enthält die relevanten Informationen zu Ihrem Thema, auf einfache und verständliche Weise zusammengefasst. Als Ergänzung zu Ihren kurzen Zusammenfassungs-Texten sollten Sie Bilder, Grafiken und/oder Diagramme erstellen. Das Fact Sheet enthält zudem kontroverse Fragestellungen, welche die Grundlage für die Gruppendiskussionen (s. Abschnitt \ref{subsec:ind}) bilden.

\textbf{Bewertungskriterien}:
\begin{itemize}
	\item (2 Pt) Inhaltlich korrekte und umfassende Zusammenfassung des Kapitels
	\item (1 Pt) Mindestens eine selbst erstellte \textbf{Illustration} (Bild, Diagramm, Grafik, usw.), die Ihr Factsheet sinnvoll ergänzt
	\item (1 Pt) Formelle Kriterien
	\begin{itemize}
		\item Abgabeformat: PDF
		\item Dateiname: ``\texttt{[Klassenname] - K[Kapitelnummer] [Kapitelname] Factsheet.pdf}'' (z.B. ``\texttt{2d - K2 Daten Factsheet.pdf}'')
		\item 1-2 A4-Seiten
		\item Korrekte Sprache, vollständige Sätze
		\item Schriftgrösse ca. 11 Punkte
		\item Vor- und Nachnamen aller Gruppenmitglieder im PDF enthalten
	\end{itemize}
	\item (1 Pt) Formulieren Sie am Ende Ihrer Zusammenfassung mindestens drei interessante, reichhaltige Diskussions-Fragen (die nicht einfach mit ja / nein beantwortet werden könnne).
\end{itemize}

\newpage
\subsection{Individueller Teil}
\label{subsec:ind}

Sie führen einen Vortrag einzeln zu Ihrem Unter-Thema Ihrer Wahl durch (s. Abschnitt \ref{sec:themen}). In Ausnahmefällen können Sie die Präsentation in Absprache mit der Lehrperson zu zweit halten, dies muss jedoch durch den grösseren Umfang des Themas begründet sein (s. Bewertungskriterien unten). Als Begleitmaterial zum Vortrag erstellen Sie eine Präsentation sowie eine Zusammenfassung, welche Sie vorgängig abgeben.

\subsubsection{Präsentation}

Bewertungskriterien für den Vortrag:

\begin{itemize}
	\item Dauer der Präsentation: \textbf{10-15 Minuten} (falls zu zweit: 15-25 Minuten), inklusive Diskussion (s. Abschnitt untenhalb dieser Liste)
	\item Der Inhalt Ihrer Präsentation soll eine didaktisch hochwertige Zusammenfassung eines Themas Ihrer Wahl aus dem Buch von Hannah Fry sein (s. Abschnitt \ref{sec:themen}).
	\item Tragen Sie sich für ein Thema Ihrer Wahl ein (\href{https://eduzh-my.sharepoint.com/:x:/g/personal/cyril_wendl_edu_zh_ch/EScHfDarDeVHhAJB3WPCghsBYa63XpdaEkcrZk5dOiQiVA?e=kHy0bM}{Link}). Falls es mehrere InteressentInnen für ein Thema gibt, sprechen Sie sich bitte untereinander ab, oder kontaktieren Sie die Lehrperson, um eine mögliche Aufteilung des Themas in mehrere Unter-Themen zu besprechen.
	\item Die detaillierten Bewertungskriterien finden Sie auf \href{https://moodle.ksimlee.ch/mod/assign/view.php?id=32090}{Moodle}.
	\item Falls Sie den Vortrag zu zweit halten, sollten Sie sich so aufteilen, dass Sprechzeit und Umfang der jeweiligen Präsentations-Teile in etwa gleich zwischen beiden Personen aufgeteilt sind.
\end{itemize}

Im Anschluss an den Vortrag findet kurze Diskussionsrunde in der Klasse statt, welche durch die Vortragenden Personen moderiert wird. Dabei sollten Sie 2-3 möglichst kontroverse und interessante Fragen formulieren, welche Sie in der Klasse diskutieren. Die Fragen sollten nicht bereits in vorherigen Vorträgen gestellt worden sein.

\subsubsection{Zusammenfassung}
Zusätzlich zum Vortrag verfassen Sie eine Zusammenfassung des von Ihnen gewählten Themas sowie ein Peer Review (=Feedback) einer anderen Zusammenfassung. 

Bewertungs-Kriterien für die Zusammenfassung sind:
\begin{itemize}
	\item (1 Pt) \textbf{Kontext}: Einbettung Ihres Unter-Themas in das Kapitel
	\item (2 Pt) \textbf{Beantwortung der Fragen zu Ihrem Thema} aus Abschnitt \ref{sec:themen}
	\item (1 Pt) Präsentation Ihrer \textbf{Recherche}-Ergebnis (s. Fragen in Abschnitt \ref{sec:themen})
	\item (1 Pt) Mindestens eine sinnvolle und aufschlussreiche \textbf{Illustration} (Bild, Diagramm, Grafik, usw.)
	\item (1 Pt) \textbf{Formelle Kriteren}
	\begin{itemize}
		\item Abgabeformat: PDF
		\item Länge: 1-2 A4-Seiten (falls zu zweit: 2-4 Seiten)
		\item Korrekte Sprache, vollständige Sätze
		\item Schriftgrösse ca. 11 Punkte
	\end{itemize}
\end{itemize}

Das Peer-Review sollte aus einem konstruktiven und ausführlichen (mind. eine halbe A4-Seite) Kommentar bestehen, der auf alle obigen Punkte eingeht (Kontext, Inhalt, Recherche, Illustration, formelle Kriterien).
