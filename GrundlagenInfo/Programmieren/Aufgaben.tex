\begin{frame}[fragile]{Aufgaben lösen \emoji{nerd-face}\emoji{books}}
  Herr Knuth hat ein eigenes Programmier-Lehrmittel entwickelt. Die darin enthaltenen Aufgaben enthalten jeweils eine Zeitangabe (benötigte Zeit um die Aufgabe zu lösen). Herr Knuth lässt Sie aus dem Buch die Aufgaben A.1 (13'), A.2 (18'), A.3 (11') und A.4 (5') daraus lösen. Die Lektion dauert 60 Minuten.\\
	
	Aus eigener Erfahrung wissen Sie, dass Sie, wenn Sie nicht ausgeschlafen sind, etwa doppelt so lange brauchen wie im Buch angegeben. Ebenso wissen Sie, dass Sie doppelt so lange brauchen, wenn Sie die Aufgabe nicht genau lesen.

	def benote_Informatiklehrperson(note):
	    if note == 100:
	        print("Echt jetzt? Vielen Dank! :)")
	        
	note = input("Benoten Sie die Lehrperson (0-100).")
	benote_Informatiklehrperson(note)

\begin{frame}[fragile]{Boolsche Ausdrücke}
\begin{itemize}
    \item \tib{Boolsche Ausdrücke} sind Ausdrücke, die genau zwei verschiedene Werte annehmen können: entweder \textit{wahr} (\pythoninline{True}) oder \textit{falsch} (\pythoninline{False}). Beispiele:
    \begin{itemize}
        \item \pythoninline{3 == 5 # False}
        \item \pythoninline{(9 + 3) < (2 * 8) # True}
    \end{itemize}
    \item Weitere boolsche Relationen:
    \begin{table}[H]
        \centering
        \begin{tabular}{r|l}
        \textbf{Operator in Python} & \textbf{Mathematische Bedeutung} \\\midrule 
             \pythoninline{==} & gleich ($=$)\\
             \pythoninline{!=} & ungleich ($\neq$) \\
             \pythoninline{<} & kleiner ($<$)\\
             \pythoninline{<=} & kleiner oder gleich ($\leq$) \\
             \pythoninline{>} & grösser ($>$) \\
             \pythoninline{>=} & grösser oder gleich ($\geq$)
        \end{tabular}
    \end{table}
    \item Verwendungsart
\begin{lstlisting}[style=TigerJython]
if BOOLSCHER_AUSDRUCK:
    # dieser Code wir höchstens dann ausgeführt, wenn 
    # BOOLSCHER_AUSDRUCK wahr ist (True)
\end{lstlisting}
\end{itemize}
   
\end{frame}
