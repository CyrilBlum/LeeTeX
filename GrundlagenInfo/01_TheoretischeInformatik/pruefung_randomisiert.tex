\question[2]
Gegeben ist eine Python Funktion \pythoninline{sieb(n)}, welche für eine gegebene natürliche Zahl $n$ eine Liste aller Primzahlen $\leq n$ ausgeben (printen) soll. Die in \cref{lst:sieb} gezeigte Funktion ist nicht korrekt. Verbessern Sie die Funktion, sodass sie die gewünschte Tätigkeit ausführt.
\begin{lstlisting}[language=Python,caption=(noch nicht korrektes) Sieb des Eratosthenes,numbers=none,label=lst:sieb]
import math

def sieb(n):
    primzahlen = []
    nicht_gestrichen = (n + 1) * [True]
    sqrtn = int(math.sqrt(n))




    i = 2
    while i <= sqrtn:
        if nicht_gestrichen[i]:
            primzahlen.append(i)
            j = 2 * i





            while j <= n:
                nicht_gestrichen[j] = False
                j += i
        i += 1






    print(primzahlen)
\end{lstlisting}
