\begin{myexercise}
Suchen Sie sich in 3er- oder 4er-Gruppen eine Stadt aus, die Sie im Rahmen der nächsten 7 Doppellektionen vertiefen werden:
\begin{itemize}
\item Basel (\& Laufen)
\item Bern (\& Huttwil)
\item Genf (\& Allaman)
\item Andere Stadt (\& Dorf) nach Wunsch
\end{itemize}
Erarbeiten Sie ein Plakat auf nächstes Mal, in welcher Sie die unterschiedlichen Definitionen dieser Städte ausarbeiten (die ``Dörfer'' in den Klammern werden wir erst später anschauen).

Stellen Sie Folgendes dar:
\begin{itemize}
\item \textbf{Statistische Definition}: Einwohnerzahl, Einwohnerdichte, sowie andere relevante Kennzahlen
\item \textbf{Historische Definition}: Historische Entstehung, Spezialrechte etc.
\item \textbf{Subjektive Definition}: Welche städtischen Aspekte verbinden \textit{Sie} mit dieser Stadt? Weshalb finden Sie (nicht), dass es eine Stadt ist?
\item \textbf{Funktionale Definition}: Suchen Sie ein Luftbild Ihrer Stadt und annotieren Sie darauf die \textbf{Grunddaseinsfunktionen}!
\end{itemize}

\textbf{Formales}: Mindestens 2 Bilder, mindestens 7 Grunddaseinsfunktionen.

\textbf{Nützliche Ressourcen:}
\begin{itemize}
\item \href{https://www.atlas.bfs.admin.ch/maps/13/map/mapIdOnly/27621_de.html}{Statistiken des BFS}, sowie weitere Karten zu Bevölkerung (Anzahl, Dichte etc.) auf derselben Webseite.
\item \href{https://www.lds.geo.uzh.ch/de/Alle-Orte.html}{Städte-Luftbilder des Universität Zürich}
\item \href{https://hls-dhs-dss.ch/de/articles/000209/2016-11-10/}{Historisches Lexikon der Schweiz, z.B. zur Stadt Bern}
\end{itemize}
