\section*{Administratives}

\section*{Thema und Inhalt}
Das Modul soll verschiedene Themen des Themenkreises rund um das \textit{Web} behandeln. 
Zu diesen Themen gehören unter anderem:
\begin{itemize}
  \item Erstellung und Gestaltung einer eigenen Webseite über ein Thema rund um das Internet
  \item Funktionsweise und Geschichte des Internets
  \item Hacking und Sicherheitslücken
  \item Datenverwaltung und Datenanalyse
  \item Malware
\end{itemize}

\section*{Beschreibung}

In diesem Modul erstellen Sie eine eigene Webseite und lernen dabei das Internet, dessen Ursprünge, Nutzen und Gefahren kennen. Nebst der Erstellung einer eigenen Webseite beschäftigen Sie sich unter anderem mit den realen Gefahren, ausgehend von Sicherheitsrisiken, Sicherheitslücken, Malware, Eindringlingen und Hacking. Dabei erlernen Sie ebenfalls \textit{best practices} in Bezug auf die Verwaltung Ihrer eigenen Daten, damit Sie nicht selbst Opfer eines geplanten Angriffs werden. Nicht zuletzt werden wir im Rahmen einer Exkursion eine Firma kennenlernen, welche die physische Infrastruktur hinter dem Internet bereitstellt. Die Interessenwoche steht sowohl für erste als auch für zweite Klassen offen. Wir setzen keine besonderen Vorkenntnisse voraus.

\subsection*{Arbeitsaufträge ``Abschlussreferate Internet-Themen''}
Im Rahmen dieses Moduls werden die SuS eine eigene Webseite zu einem vorgegebenen Thema rund um Informatik erstellen. Zu Beginn wird eine Sammlung von Artikeln zu ausgewählten Themen präsentiert und zur Verfügung gestellt. Mögliche Themen umfassen beispielsweise TikTok, soziale Medien, Sicherheitsrisiken, Umweltprobleme der Servers oder die Funktionsweise des Internets. Die SuS erhalten danach kurz Zeit, um sich in das Thema einzuarbeiten und erstellen danach selber eine Webseite, in der sie ihre Erkenntnisse zum Thema präsentieren.

\subsection*{Einführung HTML \& CSS}
Die SuS werden in die Grundlagen der Web-Programmierung mit HTML und CSS eingeführt. Hierzu werden folgende benötigte Tools gleich zu Beginn gemeinsam installiert:
\begin{itemize}
\item \textbf{LAMPP / XAMPP}: Eigener Webserver (\texttt{localhost}), auf dem Webseiten gehostet werden können
\item \textbf{VSCodium}: IDE zum Schreiben von HTML-Code
\end{itemize}

Nach der Installation der benötigten Tools erstellen wir eine eigene Webseite zum Thema, welches die SuS im Teil ``Abschlussreferate Internet-Themen'' ausgewählt haben. Dazu werden folgende Tutorials zur Verfügung gestellt:

\begin{itemize}
\item Struktur der Webseite: HTML
\begin{itemize}
\item \url{https://informatik.mygymer.ch/base/?page=network/5-web/1-0-html}
\item \url{https://www.w3schools.com/html/}
\end{itemize}
\item Darstellung der Webseite: CSS
\begin{itemize}
\item \url{https://informatik.mygymer.ch/base/?page=network/5-web/2-0-css}
\item \url{https://www.w3schools.com/css/default.asp}
\end{itemize}
\item Interaktivität (für Schnelle): JavaScript
\end{itemize}

Die SuS sollen sich selbstständig und mit der Hilfe der Lehrperson mit den neuen Themen auseinandersetzen. Sporadisch werden von den LP neue Inputs zu HTML und CSS gegeben, um sicherzustellen, dass alle SuS ein Mindest-Niveau erreichen.

Im Block ``Abschlussreferate Internet-Themen'' präsentieren die SuS ihre Webseiten sowie ihre inhaltlichen Erkenntnisse.

Die Interessenwoche wird abgerundet mit einem gemeinsamen Besuch im Internet-Museum ``Enter'' in Solothurn.
