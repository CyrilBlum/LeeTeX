# Zugriff auf kompilierte Skripte und Slides

Alle kompilierten Dateien sind über files.in-form-atik.ch verfügbar.

Direkter Einstieg:

https://files.in-form-atik.ch

Dort finden Sie die bereitgestellten PDF-Dateien für Skripte und Slides in den entsprechenden Ordnern.

Falls Sie eine Datei nicht finden, prüfen Sie zuerst, ob Sie im richtigen Fachbereich und im passenden Unterordner sind. Aktualisieren Sie danach die Seite im Browser.

Wenn weiterhin eine Datei fehlt, ist sie möglicherweise noch nicht hochgeladen oder wurde umbenannt.
