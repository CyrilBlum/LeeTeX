\chapter{Datenstrukturen}
\section{Listen}
Bisher haben wir uns mit Variablen beschäftigt, die nur einen Wert speichern können::
\begin{itemize}
    \item \lstinline|x = 3| (eine Zahl / \enquote{integer})
    \item \lstinline|name = "Hallo"| (einen Text /  \enquote{string})
    \item \lstinline|ist_wahr = True| (einen Wahrheitswert / \enquote{boolean})
\end{itemize}

In der Informatik ist es jedoch oft nötig, mit vielen Werten gleichzeitig arbeiten zu können, gerade im Kontext von \textit{Big Data}. Eine Möglichkeit, in Python mit vielen Werten zu arbeiten, sind \textbf{Listen}.

\begin{mydefinition}
Eine \textbf{Liste} ist eine Sammlung von Werten, die in einer Variablen gespeichert werden können. Eine Liste kann beliebig viele Werte enthalten, und diese Werte können von einem beliebigen Typ \enquote{integer}, \enquote{string} oder \enquote{boolean} sein.
\end{mydefinition}

\begin{myexample}[Listen erstellen]
Folgendes Beispiel zeigt, wie eine Liste in Python definiert wird. Wir können dabei, wie auch bei anderen Variablentypen, beliebige Namen verwenden. Die Inhalte der Liste werden in eckigen Klammern \lstinline|[]| geschrieben und die einzelnen Werte werden durch Kommas \lstinline|,| getrennt.
\begin{lstlisting}
# Liste mit Zahlen  
l1 = [1, 2, 3, 4, 5]
# Liste mit Texten
l2 = ["Hallo", "Welt", "Python"]
# Liste mit Wahrheitswerten
l3 = [True, False, True]
# Liste mit gemischten Werten
l4 = [1, "Hallo", True, 3.14]
\end{lstlisting}
\end{myexample}
\begin{myexample}[Zugriff auf Listen]
Wenn wir auf die einzelnen Werte in der Liste zugreifen möchten, können wir dies mit dem Index tun. Der Index ist eine Zahl, die angibt, an welcher Stelle sich der Wert in der Liste befindet. Der Index beginnt bei 0, das heisst, der erste Wert in der Liste hat den Index 0, der zweite Wert hat den Index 1 und so weiter.
\begin{lstlisting}
# Zugriff auf den ersten Wert in der Liste
print(l1[0]) # gibt 1 aus
# Zugriff auf den zweiten Wert in der Liste
print(l1[1]) # gibt 2 aus
# usw.
\end{lstlisting}
\end{myexample}
\begin{myexample}[Listen-Werte verändern]
Um den Wert an einer bestimmten Stelle in der Liste zu ändern, können wir ebenfalls den Index verwenden. Wir können den Wert an dieser Stelle einfach durch einen neuen Wert ersetzen.
\begin{lstlisting}
# Ändern des Wertes an der Stelle 0 in der Liste l1
l1[0] = 10
print(l1) # gibt [10, 2, 3, 4, 5] aus
\end{lstlisting}
\end{myexample}
\begin{myexample}[Länge einer Liste]
Der Befehl \lstinline|len(liste)| gibt die Länge der Liste zurück, also die Anzahl der Werte, die in der Liste gespeichert sind. Dies ist nützlich, wenn wir wissen möchten, wie viele Werte in der Liste enthalten sind.
\end{myexample}

def skalarprodukt(m, p):

skalarprodukt(m, p)
\end{lstlisting}
    \end{myanswer}

\begin{mychallenge}
    Schreiben Sie eine Funktion, die überprüft, ob eine Liste von Zahlen sortiert ist oder nicht. Falls die Liste sortiert ist, soll die Definition den Wert \lstinline|True| zurückgeben, ansonsten den Wert \lstinline|False|.

    Tipps:
    \begin{enumerate}
        \item Gehen Sie jedes Element der Liste mit einer Schleife durch, und überprüfen Sie, dass das Element grösser oder gleich dem vorigen Element ist.
        \item \lstinline|return| bricht die Funktion ab und gibt einen Wert an das Hauptprogramm zurück. Sie können also, sobald eine Zahl in falscher Reihenfolge gefunden worden ist, direkt \lstinline|False| zurückgeben.
        \item Falls Sie nie ein falsches Element gefunden haben, geben Sie am Ende der Defintion \lstinline|True| zurück.
    \end{enumerate}
    \begin{myanswer}
        \begin{lstlisting}
def ist_sortiert(liste):
    i = 1
    while i < len(liste):
        if liste[i] < liste[i-1]:
            return False
        i += 1
    return True
\end{lstlisting}
    \end{myanswer}
\end{mychallenge}

\subsection{Algorithmen}
\subsubsection{Sortier-Algorithmen}
Eine der häufigsten Anwendungen in der Informatik ist das Sortieren von Daten. Es gibt viele verschiedene Algorithmen, um Daten zu sortieren, und jeder Algorithmus hat seine eigenen Vor- und Nachteile, insbesondere hinsichtlich der Geschwindigkeit und der benötigten Rechenleistung. Im Folgenden wird der Bubble-Sort-Algorithmus vorgestellt, der eine einfache Methode ist, um Daten zu sortieren. Der Bubble-Sort-Algorithmus funktioniert, indem er die Liste von Werten durchläuft und benachbarte Werte vergleicht. Wenn ein Wert grösser ist als der nächste Wert, werden die beiden Werte vertauscht. Dieser Vorgang wird so lange wiederholt, bis die gesamte Liste sortiert ist. Die ersten zwölf Schritte des Bubble-Sort-Algorithmus sind in \cref{fig:bubble-sort} dargestellt. Der Algorithmus wird so lange wiederholt, bis die gesamte Liste sortiert ist. Der Algorithmus hat eine Zeitkomplexität von $O(n^2)$, was bedeutet, dass die Laufzeit des Algorithmus quadratisch mit der Anzahl der Werte in der Liste wächst. Dies macht den Bubble-Sort-Algorithmus für grosse Datenmengen ineffizient.

\subsubsection{Such-Algorithmen}
In der Informatik stellt sich häufig die Frage, wie man möglichst schnell herausfinden kann, ob eine bestimmte Zahl in einer Liste enthalten ist.

Ist die Liste unsortiert, wie zum Beispiel bei \lstinline|x = [5, 3, 8, 20, 2, 10]|, bleibt uns nichts anderes übrig, als jedes Element der Liste einzeln zu überprüfen. Dies entspricht einer linearen Suche, bei der im ungünstigsten Fall alle Elemente betrachtet werden müssen.

Ist die Liste jedoch bereits sortiert, wie zum Beispiel bei \lstinline|x = [2, 3, 5, 8, 10, 20]|, können wir effizientere Suchverfahren anwenden. Dies lässt sich mit der Suche nach einem Gegenstand in einem Koffer vergleichen. Wenn der Koffer unordentlich gepackt ist, müssen wir jeden einzelnen Gegenstand herausnehmen, um den gesuchten zu finden. Wenn der Koffer jedoch ordentlich gepackt ist, können wir die Gegenstände viel schneller finden (s. \cref{fig:koffer-suche}).

Ein besonders schneller Algorithmus ist die sogenannte \textbf{binäre Suche}, bei der die Liste immer wieder halbiert wird, um das gesuchte Element zu finden.

Das Suchen bezeichnet allgemein den Vorgang, ein bestimmtes Element in einer Datenmenge möglichst effizient zu finden. Im Folgenden lernen wir die binäre Suche als effizienten Such-Algorithmus für sortierte Listen kennen.

\subsection{Listen verändern}
\subsubsection{Elemente zu Listen hinzufügen und entfernen}

In Python gibt es verschiedene Methoden, um Listen dynamisch zu verändern. Die wichtigsten Methoden sind \lstinline|append|, \lstinline|insert| und \lstinline|pop|.

\begin{mydefinition}
\begin{itemize}
    \item \lstinline|liste.append(wert)| fügt \textbf{am Ende} der Liste einen neuen Wert hinzu.
    \item \lstinline|liste.insert(index, wert)| fügt an der angegebenen Position (\texttt{index}) einen neuen Wert ein. Alle nachfolgenden Elemente werden nach rechts verschoben.
    \item \lstinline|liste.pop(index)| entfernt das Element an der angegebenen Position (\texttt{index}) und gibt es zurück. Wird kein Index angegeben, wird das letzte Element entfernt.
\end{itemize}
\end{mydefinition}

\begin{myexample}[Elemente hinzufügen und entfernen]
Folgender Code zeigt auf, wie Listen in Python verändert werden können. Wir erstellen eine Liste von Zahlen und fügen dann neue Zahlen hinzu, entfernen das letzte Element und das erste Element. Die Ergebnisse werden anschliessend ausgegeben.
\begin{lstlisting}
zahlen = [1, 2, 3]
zahlen.append(4)        # [1, 2, 3, 4]
zahlen.insert(1, 10)    # [1, 10, 2, 3, 4]
letztes = zahlen.pop()  # entfernt 4, jetzt [1, 10, 2, 3]
erstes = zahlen.pop(0)  # entfernt 1, jetzt [10, 2, 3]
print(zahlen)           # [10, 2, 3]
print(letztes)          # 4
print(erstes)           # 1
\end{lstlisting}
\end{myexample}

\begin{myexercise}
    Fügen Sie der Liste \lstinline|fruechte = ["Apfel", "Banane"]| zuerst \lstinline|"Orange"| am Ende hinzu, dann \lstinline|"Kiwi"| an Position 1. Entfernen Sie danach das erste Element und geben Sie die Liste aus.

    \begin{myanswer}
        \begin{lstlisting}
fruechte = ["Apfel", "Banane"]
fruechte.append("Orange")      # ["Apfel", "Banane", "Orange"]
fruechte.insert(1, "Kiwi")    # ["Apfel", "Kiwi", "Banane", "Orange"]
fruechte.pop(0)               # entfernt "Apfel"
print(fruechte)               # ["Kiwi", "Banane", "Orange"]
\end{lstlisting}
    \end{myanswer}
\end{myexercise}

\begin{myexercise}
    Erstellen Sie eine leere Liste \lstinline|zahlen|. Fügen Sie mit einer Schleife die Zahlen 1 bis 5 mit \lstinline|append| hinzu. Entfernen Sie dann das Element an Position 2 mit \lstinline|pop| und geben Sie die Liste aus.

    \begin{myanswer}
        \begin{lstlisting}
zahlen = []
for i in range(1, 6):
    zahlen.append(i)
zahlen.pop(2)  # entfernt die 3
print(zahlen)  # [1, 2, 4, 5]
\end{lstlisting}
    \end{myanswer}
\end{myexercise}

farben = ["rot", "blau", "grün"]

\section{Dictionaries}
\subsection{Dictionaries}
In der Informatik werden Daten oft nicht nur als Listen, sondern auch als sogenannte \textbf{Dictionaries} (Wörterbücher) gespeichert. Ein Dictionary ist eine Sammlung von Schlüssel-Wert-Paaren. Anders als bei Listen werden die Werte nicht durch einen Index, sondern durch einen eindeutigen \textbf{Schlüssel} (key) angesprochen. Dies ist vielfach praktischer als die Speicherung in Listen, da man direkt mit einem Begriff auf die Werte zugreifen kann, ohne die Position im Array kennen zu müssen.

\begin{mydefinition}
Ein \textbf{Dictionary} ist eine Datenstruktur, die jedem Schlüssel (\textit{key}) einen Wert (\textit{value}) zuordnet. Die Schlüssel müssen eindeutig sein und können z.B. Zahlen oder Zeichenketten sein.
\end{mydefinition}

\begin{myexample}[Dictionary für Kontaktdaten]
Ein typisches Beispiel für ein Dictionary ist ein Adressbuch, in dem zu jedem Namen die Telefonnummer gespeichert ist:

\begin{lstlisting}
telefonbuch = {
    "Anna": "079 123 45 67",
    "Ben": "078 987 65 43",
    "Clara": "077 555 44 33"
}
print(telefonbuch["Anna"])  # gibt "079 123 45 67" aus
\end{lstlisting}
\end{myexample}

\begin{myexample}[Dictionary für Produktpreise]
Auch in einem Online-Shop werden Produkte oft mit ihren Preisen als Dictionary gespeichert:
\begin{lstlisting}
preise = {
    "Apfel": 0.80,
    "Banane": 0.50,
    "Brot": 2.50
}
print(preise["Brot"])  # gibt 2.5 aus
\end{lstlisting}
\end{myexample}

\begin{myexample}[Werte hinzufügen und ändern]
Sie können einem Dictionary neue Schlüssel-Wert-Paare hinzufügen oder bestehende Werte ändern:
\begin{lstlisting}
preise["Milch"] = 1.60         # neues Produkt hinzufügen
preise["Apfel"] = 0.90         # Preis ändern
print(preise)
\end{lstlisting}
\end{myexample}

\begin{myexample}[Alle Schlüssel und Werte durchgehen]
Mit einer Schleife können Sie alle Einträge eines Dictionaries durchgehen:
\begin{lstlisting}
for produkt in preise:
    print(produkt, "kostet", preise[produkt], "Franken")
\end{lstlisting}
\end{myexample}

\begin{myexercise}
    Erstellen Sie ein Dictionary \lstinline|noten|, das die Noten von drei Schülern speichert: \lstinline|"Lea"| (5.5), \lstinline|"Tim"| (4.0), \lstinline|"Sara"| (6.0). Geben Sie die Note von \lstinline|"Sara"| aus.

    \begin{myanswer}
        \begin{lstlisting}
noten = {"Lea": 5.5, "Tim": 4.0, "Sara": 6.0}
print(noten["Sara"])  # gibt 6.0 aus
\end{lstlisting}
    \end{myanswer}
\end{myexercise}

\begin{myexercise}
    Fügen Sie dem Dictionary \lstinline|noten| aus der vorherigen Aufgabe einen neuen Schüler \enquote{Alex} mit der Note 5.0 hinzu. Ändern Sie Tims Note auf 4.5 und geben Sie das gesamte Dictionary aus.

    \begin{myanswer}
        \begin{lstlisting}
noten["Alex"] = 5.0
noten["Tim"] = 4.5
print(noten)
\end{lstlisting}
    \end{myanswer}
\end{myexercise}

\begin{myexercise}
    Sie verwalten die Lagerbestände eines kleinen Geschäfts. Erstellen Sie ein Dictionary \lstinline|lager| mit den Produkten \lstinline|"Cola"| (10 Stück), \lstinline|"Fanta"| (5 Stück) und \lstinline|"Wasser"| (20 Stück). Schreiben Sie ein Programm, das die Anzahl der Flaschen \lstinline|"Fanta"| um 2 reduziert (z.B. durch Verkauf) und das Ergebnis ausgibt.

    \begin{myanswer}
        \begin{lstlisting}
lager = {"Cola": 10, "Fanta": 5, "Wasser": 20}
lager["Fanta"] -= 2
print(lager)
\end{lstlisting}
    \end{myanswer}
\end{myexercise}

\begin{myexercise}
    Erstellen Sie ein Dictionary, das für verschiedene Länder die jeweilige Hauptstadt speichert (z.B. "Schweiz": "Bern", "Deutschland": "Berlin", "Frankreich": "Paris"). Lassen Sie den Benutzer mit \lstinline|input()| nach einer Hauptstadt fragen und geben Sie diese aus.

    \begin{myanswer}
        \begin{lstlisting}
hauptstaedte = {
    "Schweiz": "Bern",
    "Deutschland": "Berlin",
    "Frankreich": "Paris"
}
land = input("Land eingeben: ")
if land in hauptstaedte:
    print("Die Hauptstadt ist:", hauptstaedte[land])
else:
    print("Land nicht gefunden.")
\end{lstlisting}
    \end{myanswer}
\end{myexercise}

Dictionaries sind besonders nützlich, wenn Sie Daten eindeutig zuordnen und schnell darauf zugreifen möchten, z.B. für Benutzerprofile, Konfigurationen, Übersetzungen oder Lagerbestände.
