\begin{frame}{Lernziele}
    \begin{itemize}
        \item Sie können erklären, was eine Funktion in der Programmierung ist.
        \item Sie können die Vorteile von Funktionen erläutern (z.B. Wiederverwendbarkeit, Modularität).
        \item Sie können die Syntax zur Definition einer eigenen Funktion beschreiben.
        \item Sie können eine selbst definierte Funktion aufrufen.
    \end{itemize}
\end{frame}
